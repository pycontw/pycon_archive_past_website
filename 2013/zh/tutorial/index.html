PyConTW 2013 先修課程

這是首次在 PyConTW 安排收費的課程時段，以 Python 入門為課程內容，目標之一是讓參與 PyConTW 的聽眾，事先學到更多基本知識，這樣，在活動期間更容易吸收議程內容，更容易促成社群交流效果。

時間

2013 / 05 / 24 (五) 正式議程開始前一天

  • 09:00-09:30 學員報到
  • 09:30-12:30 上午課程
  • 12:30-14:00 午休時間 (自行外出用餐)
  • 14:00-17:00 下午課程

上課地點

地址: 台北市忠孝東路四段178號4樓 (聯成電腦忠孝分校)

鄰近捷運忠孝敦化站

緊急聯絡電話: 02-2740-3862

報名方式

請先完成年會門票的報名，持年會報名序號，到 registrano 報名先修課程。

適合對象

屬於入門課程，適合初學者，之前無須具備程式經驗。投影片以英文編寫，使用中文解說，附有 lab 練習。

課程效益

透過投影片進行簡要說明與介紹，而後進行實作，以此模式不斷進行直到課程結束。協助學員在課程結束之後，能對 Python 的核心文化與生態有所瞭解。

課程目標

  • 學習 Python 入門語法及工具
  • 認識 Python 核心與生態 (語言、程式庫與文化)

課程內容

課程大綱包括：

  • Python 版本的選擇與安裝
  • 簡介 Unicode 支援
  • 基本輸入輸出
  • 內建型態
  • if、for、while 與 for Comprehensions
  • 函數、模組、類別與套件
  • 社群
  • 文件
  • 資料處理函式
  • 物件序列化
  • 資料庫程式
  • Django
  • 測試
  • 其他學習資源

課程穿插頻繁實作，讓學員從實作中驗證觀念。
