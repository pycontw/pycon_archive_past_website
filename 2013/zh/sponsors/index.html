Microsoft

微軟公司成立於 1975 年，多年來在全球個人電腦與商用軟體、服務與網際網路技術上居領導地位。微軟公司致力於提供各種產品與服務，讓人們在任何時間、任何地點、使用任何裝置，都能輕鬆取得資訊。

WebFaction

WebFaction 是技術領先的 Python 主機代管服務供應商。 除了提供快速可靠的主機代管服務以及先進彈性的設定， 我們的主機實體位置還遍及亞洲、歐洲以及美國。 我們所有的伺服器裡都已內建了八個不同版本的 Python (從 Python 2.4 到 Python 3.3)，且每個版本都附上了好用的模組。此外，我們也提供一鍵設定 Django, Pyramid, Zope/Plone 及其它服務。 WebFaction 很榮幸能支持 Python 社群，並提供 6 個月的免費主機服務給所有報名參加 PyCon Taiwan 2013 的社群朋友！

http://www.webfaction.com

zillians

奇群科技，成立於2008年。是個有各種不同背景的熱血開發者團隊，共同目標為建立終極的雲端運算平台服務。我們專精於高效能計算系統、分散式平行運算系統、及編譯器開發。 我們的願景是要成為台灣雲端運算領導供應商，提供世界級的雲端平技術及服務。藉由HPC技術，建構新一代雲端架構來開發高效能、即時的雲端應用。

http://www.zillians.com

自由軟體鑄造場

中央研究院資訊科學研究所於民國92年接受經濟部工業局「自由軟體產業推動計畫」的補助，成立自由軟體鑄造場 (Open Source Software Foundry，以下簡稱 OSSF )，以擴大自由軟體在臺灣的應用幅度與研發能量。OSSF 重點任務為培植本土自由軟體社群與人才，工作內容包括 (一) 推廣自由軟體概念；(二) 集結國內自由軟體社群；(三) 促進臺灣人才培育與產業合作；(四) 與國際趨勢交流接軌。

OSSF 計畫並透過 OpenFoundry 平台，提供使用者從事自由軟體專案開發時，所需要的重要工具與服務。

http://www.openfoundry.org/

StorySense

StorySense 是一家以人工智慧技術為背景的公司。本團隊以 "Making Grandpa-Friendly Computers"（創造阿公阿嬤都覺得簡單好用的智慧型軟體）為願景，開發具語言分析及故事理解能力的應用軟體，期許能夠幫助使用者消化整理網路上大量的資料、簡化使用流程，並創造簡單方便的使用經驗。日前發表的 WhatsTheNumber 手機查號軟體，目前已在 Apple 及 Android 兩個 App Store 供使用者下載，並開放超過一百個國家的使用者使用，累積下載人次已逾六十萬人次。我們的辦公室坐落在美麗的仁愛路上、國父紀念館旁，希望團隊裡的每個人都能有一個愉快的工作環境和平衡豐富的生活。

http://www.storysensecomputing.com/

Pinkoi

Pinkoi 是全台灣最大的設計品社群購物平台，也是亞洲最大的設計師社群，目前累計有兩千多個品牌、七千多位駐站設計師，近五萬件的獨特設計商品。在 Pinkoi，你可以買設計、品設計、賣設計，我們希望讓設計回歸生活、提供美好的購物體驗，讓每一件設計品都能帶給每個人禮物般的貼心與感動。

http://www.pinkoi.com/

Wingware

Wingware 的 Wing IDE 專為 Python 設計。Wing IDE 具有強大的程式碼智慧編輯、修訂、除錯和測試功能，在強化您的生產力、減少程式錯誤之餘，還能讓您在程式碼中輕鬆地查找以瞭解不熟悉的程式。Wing IDE 可在 Windows、Linux 和 OSX 等平台運作，並且能和 Django、Turbogears、Plone、GAE、wxPython、PyQt、PySide、PyGtk、TkInter、pygame 和許多框架、模組搭配。自 1999 年起，Wingware 便致力於 Python 技術應用，更多詳情，請參訪 wingware.com。

塔圖科技

塔圖科技，2010年成立，著重研發網路廣告、圖片相關應用，曾入選appWorks育成計畫、IDEAS Show網路創意參展團隊和台大StartUp。

目前產品有圖片標籤工具和準買家廣告兩項，準買家廣告為台灣第一個即時廣告競價系統(RTB)，也是第一個專門針對電子商務廠商再行銷(Remarketing)的系統。

我們希望提供電子商務廠商更有效率、收益更高的廣告解決方案。

http://www.tagtoo.com.tw/

杏昕科技

杏昕科技股份有限公司主要業務為醫療器材設計服務，2013 年連續獲得德國 iF 設計獎以及德國紅點 (red dot) 設計獎。「輕鬆使用，完整照護」是 AcoMo Technology 研發團隊致力達到的設計目標。而開發一系列標誌著「台灣製造 MIT」的優質精品，更是研發團隊從「心」稟持的初衷。產品範圍將涵括嬰幼兒照護所需的各個面向，以高質量的電子設計研發實力為基礎，讓新生兒父母能輕鬆的使用 AcoMo Technology 所開發的種種人性化精品，在繁瑣的家務活動中，仍能周密地給予每一個寶貴的幼小生命 24 小時 365 天的零距離照護。

http://www.acomotech.com/

聯成電腦

聯成電腦原廠授權訓練中心創立於民國80年，始終以誠懇、實在的態度致力於電腦教學，不斷追求課程創新以及教學品質提升，並多次得到全球大廠肯定，連年獲頒最佳教育訓練夥伴及授權訓練中心等獎項。包括 Microsoft、Adobe、Redhat、Corel、Celsys、Sun、Oracle、Autodesk、TQC、Wacom等皆是與聯成電腦長期密切合作的夥伴關係，20年來已為台灣培育出許多優秀的專業資訊人才。

本公司是由一群具有豐厚的專業能力及經驗的技術團隊所組成，於近二十年於全國拓點的歷程中，亦培養了具備豐富的據點拓展經驗、教室場地建置能力、軟硬體評估採購經驗、招生業務開發能力、學員認證輔考能力、講師管理與培訓能力、場地營運維護、教務管理與就業輔導機制。

http://www.lccnet.com.tw/

Sentry

Sentry 是專注於截取並收集事件記錄的平台。

在 2010 年初，孕育自 DISQUS 的 Sentry 本來是為了解決 Django 應用的例外事件而開發的。之後，Sentry 便獨立發展成支持許多主流程式語言及平台的應用。包括 Python、PHP、Java、Ruby、Node.js 甚至 JavaScript 都在支援之列。

http://getsentry.com/

友環股份有限公司

友環成立於2007年，為全球知名軟體品牌的亞洲區代理商/經銷商，以專業的技術支援、完善的咨詢服務，提供客戶多元化的軟體解決方案，並取得眾多廠商與客戶的信賴，目前更致立於推行軟體教育訓練，產品包括JIRA (Atlassian), Omnipeek (WildPackets), SpiraTest (Inflectra)..., 舉凡工研院、資策會、智易、台灣博世、長庚大學等，皆在友環專業指導下，於短時間內更了解產品的特性、提高使用效能，提昇使用者的應用能力及處理技巧。友環的客群能夠遍及公部門，上市上櫃公司、中小型企業、非營利機構、學術研究單位，是因為我們集結優質的人才，傾聽客戶的需求，尋求合適的方案，為每一位用戶打造最佳的軟體使用經驗。

http://www.linksoft.com.tw/

廣達研究院

廣達研究院 (Quanta Research Institute) 是廣達內部未來前瞻科技與產品的研發單位，更是引領雲端創新研發的火車頭。研究院透過整合國內外研發資源，以創新技術的研發與技術人才的培育，強化廣達的核心技能、創造利基產品。

廣達研究院以「人本自動化創新」為宗旨，透過貼近使用者 (end user)需求的研究、創造新的使用者經驗，統合廣達各事業群在伺服器端、無線網路端、和使用者終端的各類產品線及領先技術，讓使用者 (Anyone) 可以在任何時候 (Anytime)、任何地方 (Anywhere)，享受無間斷的雲端技術服務。

未來廣達研究院將繼續積極開發全新雲端系統與服務，並結合創新商業模式，創造市場新價值。

http://www.quantatw.com/Quanta/chinese/about/researchinstitute.aspx

協辦

國研院國網中心自由軟體實驗室

國研院國網中心自由軟體實驗室主要將高效能計算 (High Performance Computing）所累積的經驗，以自由軟體的方式呈現成果。相關的開發主要都與高效能計算有關，或者衍生出更廣泛的運用。所開發的軟體包含 DRBL (Diskless Remote Boot in Linux), Clonezilla, DRBL-Winroll, Tux2live, partclone 等，這些軟體都可以廣泛運用在高效能計算或者電腦教室中。此外，我們也提供一些鏡射 (mirror) 服務，包含主要的 GNU/Linux 套件，以及相關的自由軟體等下載服務。

本實驗室接受專案委託，包括：電腦教室 DRBL/Clonezilla 架設安裝與諮詢服務 與 Clonezilla 客製化服務

http://www.nchc.org.tw/
