PyCon Taiwan 2013 Financial Aid 計劃

2013.04.09 by registration

Everybody Pays (公平付費) 是 PyCon Taiwan 的核心價值之一。所有的會議參與者，如：講者、會議主席、工作人員、志工，皆需要自己付費，才能夠參加會議。會這麼做的原因，已於 Everybody Pays 一文寫明。

然而，在維持 Everybody Pays 此一核心價值的同時，我們也規劃了 Financial Aid (財務協助) 的方案，希望能夠協助對會議有興趣，但是資金支出較為困難的朋友來參與這次的會議。我們將採取審核制，請有此需求的朋友主動與我們聯絡，我們收到所有的申請後，會根據預算、申請者狀況等因素，決定是否給予 Financial Aid。

事實上，我們並不想對 Financial Aid 的受理對象有太多硬性的限制！是的，您若是資金支出較為困難，而您在熱愛 Python 之餘也願意為 Python 社群付出些什麼，這樣子就算是符合我們的申請標準了！但是，由於我們的 Financial Aid 經費有限，因此能夠補助的數量不多，在這種情況下我們會比較希望受理的對象大致為：「最需要被幫助的人」或「對與會者幫助最大的人」。舉例而言，若拿家庭收入欠佳的學生與已有穩定工作的工程師相比，我們會優先協助學生。更進一步地，以申請者身分來講，對 Python 社群有所貢獻者抑或這次會議的講者，都會是我們優先協助的目標。

申請 Financial Aid 的方式如下：

  1. 仍然請您依循正常管道報名參與會議，並且盡可能完成繳費。
  2. 請您寄信至 organizers@pycon.tw，簡要描述一下您的身分 (真實姓名) 與情況，並且留下聯絡方式 (email 與電話)。
  3. 我們會有專人與您聯絡，並且核對資料及了解您的狀況。
  4. Financial Aid 的受理時間為即日起至正常報名截止日。
  5. 我們會主動告知您是否有申請到 Financial Aid，若有，則使用退費的方式將費用退還給您。

我們知道每個人的經濟狀況，又或者是對 Python 社群的貢獻皆是難以比較的。而 Financial Aid 的預算又有限，無法掛保證能幫到所有人，但我們還是想要對有困難的朋友們說：「需要幫忙就說一聲吧！我們盡力！」(握拳)
