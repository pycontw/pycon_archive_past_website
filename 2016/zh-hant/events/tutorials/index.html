教學 (Tutorial)

專業課程 (Tutorial) 是 PyCon TW 2016 的一部份，採付費方式。舉辦時間會在研討會前一至三週，提供全天（6 小時）或是半天（3 小時）的課程。課程內容採取公開遴選方式徵求提案。

本次課程

  • 6/3(Fri.) 14:00~17:10 Creating python app with MySQL Router and MySQL Group Replication
  • TBD Learn to become a full-stack developer with Raspberry Pi + Python + C

    • 註：Learn to become a full-stack developer with Raspberry Pi + Python + C 目前因報名人數不足，目前開課時間待定。

售票時程

開始售票 2016 年 4 月 18 日下午 6:03

售票資訊

  • 請注意 Creating python app with MySQL Router and MySQL Group Replication 課程票僅提供給已購買 PyCon 主活動票的會眾。
  • 報名人數達 5 人即開班，未滿 5 人則流團退還全額報名費。
  • 不符資格者，大會將扣除行政費用後退餘款。
