那些年，我用 Django Admin 接的案子

語言

中文演講/英文投影片

分類

網站框架

Python 難易度

中階

投影片連結

摘要

強大的後台管理系統（Django Admin），是 Django 在眾多網站框架中脫穎而出的主因之一。
不僅擁有快速開發的特性，也保留了深度客製的彈性。
本次演講將透過接案的情境，分享如何針對不同需求，修改顯示欄位、排序、Filter、搜尋、權限管理、介面優化等需要客製化的項目。
只要學會了這些技巧，人人都可以用 Django Admin 接案！
