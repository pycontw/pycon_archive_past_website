Abstract

Motivating teenagers to learn programming can be challenging for they have short attention span and get bored easily. Fortunately, Python and contributions from the community come to the rescue.

Learning to program should be fun and engaging. With projects like Raspberry Pi, Minecraft, CheckiO, IPython Notebook, and PyCharm Edu, educators can introduce Python programming to teens using creative and captivating approaches.

Edward S. Duh

畢業於 USC 資訊工程所及 Carnegie Mellon University 企研所，工作經歷包括 IBM、華泰電子與華致資訊。目前的研究興趣著重於數據科學 (Data Science)、雲端應用 (Cloud Computing) 以及線上學習 (Online Learning)。
