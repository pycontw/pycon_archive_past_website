分享兒童程式教學經驗 Teach Kids to Learn Python

Language

Chinese talk w. Chinese slides

Category

Education

Python Level

Novice

Abstract

分享多年來兒童程式設計教學實務經驗，Python簡潔易懂的特性，加上重點式引導過程，讓學生學習後，引發對程式設計的興趣，進而真正動手寫Py Code。

Description

分享針對小六到高一設計Python入門課程，學習時Python應先具備的能力，學習Python入門的9個重點，課程式上機課練習方式，以及最重要的是如何引發學生學習的動機，培養寫程式的能力和自信心!

Eric Huang

現職為兒童程式和樂高機器人教師，創辦大拇指教室，多年來專注於國小和國中兒童程式（Python & Scratch)和樂高EV3機器人教學，希望透過創新和系統化教學可以帶出更多喜歡「動手做、動腦想」的孩子!
