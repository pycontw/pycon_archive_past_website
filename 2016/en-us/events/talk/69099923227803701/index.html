Strategies for concurrency and parallelism in Python

Language

Chinese talk w. English slides

Category

Other

Python Level

Intermediate

Slides Link

Abstract

Introduce to different strategies for task execution in python, which includes the basic synchronous model, multi-threading, multi-processing and distributed worker solutions in python. We then compare the pros and cons of these solutions, and make a brief introduction to a more scalable structure, the cloud.

洪鈺庭

Yuting Hung, a python developer, a language addict, a backpacking enthusiast, and an underwater cinematographer from Taiwan.
